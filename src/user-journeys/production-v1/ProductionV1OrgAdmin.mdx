# Production V1: Org Admin — Full Feature Set

Production RBAC instance with **Org Admin** privileges and V1 (User Access) navigation.

## Environment Configuration

- **User Role**: Organization Administrator
- **Permissions**: Full RBAC + inventory access (`rbac:*:*`, `inventory:groups:read`, `inventory:groups:write`)
- **Feature Flags**:
  - `platform.rbac.workspaces-list`: true (Workspaces page inside User Access)
  - `platform.rbac.group-service-accounts.stable`: true (current SA flag)
  - `platform.rbac.common-auth-model`: true (selectable users table)
