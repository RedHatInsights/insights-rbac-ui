# V1: Organization User (No Permissions)

Regular user with **no RBAC permissions** in V1 navigation. Can only use My User Access.

## Permission Configuration

| Permission | Has Access? |
|------------|-------------|
| `rbac:principal:read` | **No** |
| `rbac:group:read` | **No** |
| `rbac:role:read` | **No** |
| `inventory:groups:read` | **No** |
| Org Admin | **No** |

- **User Access** section — hidden (no RBAC permissions)
