# V2: Organization User (No Permissions)

Regular user with **no RBAC permissions**. Can only use the My Access page.

## Permission Configuration

| Permission | Has Access? |
|------------|-------------|
| `rbac:principal:read` | **No** |
| `rbac:group:read` | **No** |
| `rbac:role:read` | **No** |
| `inventory:groups:read` | **No** |
| Org Admin | **No** |

## V2 Navigation Context

- **My Access** — visible (always available)
- **Access Management** — hidden (no RBAC permissions)
- **Organization Management** — hidden (not org admin)

## What This Persona Can Do

- View their own access via My Access page

## What This Persona Cannot Do

- Access Users, Groups, Workspaces, or Roles pages
- See the Access Management sidebar section
- See the Organization Management sidebar section
