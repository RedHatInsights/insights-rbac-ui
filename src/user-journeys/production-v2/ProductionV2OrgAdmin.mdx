# Production V2: Org Admin — Full Feature Set

Production RBAC instance with **Org Admin** privileges and **all Management Fabric features enabled** (M1–M5).

## Environment Configuration

- **User Role**: Organization Administrator
- **Permissions**: Full RBAC access (`rbac:*:*`)
- **V2 Features Enabled**:
  - ✅ Workspace list view (M1)
  - ✅ Workspace hierarchy management — create, edit, move, delete (M2)
  - ✅ Workspace detail pages with Roles tab (M3)
  - ✅ Role bindings write — grant access wizard, remove from workspace (M4)
  - ✅ Master flag enabled (M5)
  - ✅ Access Management Navigation
  - ✅ Users and User Groups (tabbed page)
  - ✅ Common Auth Model
